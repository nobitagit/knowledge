## Unix

### How to list all active connections

```sh
sudo netstat -plunt
```

-p, --program Show the PID and name of the program to which each socket belongs.
-l, --listening Show only listening sockets.
-u UDP
-n numeric - instead of names, like ssh, http
-t TCP
